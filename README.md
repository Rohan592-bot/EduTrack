# 🎓 EduTrack — Student Performance Dashboard

EduTrack is a modern, browser-based dashboard for analyzing and visualizing student academic performance using CSV data.

Built using pure HTML, CSS, and JavaScript — no frameworks, no backend required.

---

## 🚀 Features

- 📊 **Class Overview Dashboard**
  - GPA distribution
  - Program-wise performance
  - Gender comparison
  - Trend analysis across semesters

- 👤 **Student Profiles**
  - Search and filter students
  - Tier classification (Distinction, Merit, Pass, At Risk)
  - Individual performance view

- 📈 **Comparison Tool**
  - Compare multiple students visually
  - Line charts and bar charts

- 📚 **Semester Breakdown**
  - Year-wise CGPA insights
  - Heatmaps and rankings

- 📋 **Full Data Table**
  - Sorted student list
  - Pagination support

- 📤 **CSV Support**
  - Upload student dataset
  - Export full or filtered data

- 🖨 **Report Generation**
  - Print-ready student performance reports

---

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Charts:** Chart.js  
- **Data Handling:** Vanilla JS (no libraries)
